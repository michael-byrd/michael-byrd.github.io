Linear Analysis Notes

Definition - An inner product (or scalar product) on a complex vector space $V$ is a mapping $$ (\cdot, \cdot): V \times V \to \mathbb{C} $$ such that, for all $x, y, z \in V$ and all $\lambda \in \mathbb{C},$
  • $(x,y) = \overline{(y,x)},$
  • $(\lambda x, y) = \lambda (x,y),$
  • $(x+y, z) = (x,z) + (y,z),$
  • $(x,x) > 0$ when $x \neq 0.$
An inner product space is a pair $(V, (\cdot, \cdot))$ where $V$ is a complex vector space and $(\cdot, \cdot)$ is an inner product on $V.$
Theorem - For any $x, y, z$ in an inner product space $V$ and any $\lambda \in \mathbb{C},$
  • $(x, y+z) = (x, y) + (x, z)$,
  • $(x, \lambda y) = \overline{\lambda} (x, y)$,
  • $(x, 0) = 0 = (0, x),$
  • if $(x, z) = (y, z)$ for all $z \in V$ then $x = y$.
Definition - The norm of a vector $x$ in an inner product space is defined to be $\sqrt{ (x,x) }.$ It is written $\lvert \lvert x \rvert \rvert$.
Theorem - For any $x$ in an inner product space $V$ and any $\lambda \in \mathbb{C}$
  • $\lvert \lvert x \rvert \rvert \geq 0$; $\lvert \lvert x \rvert \rvert = $ if and only if $x = 0,$
  • $\lvert \lvert \lambda x \rvert \rvert = \lvert \lambda \rvert \cdot \lvert \lvert x \rvert \rvert$
